## :clipboard: Description
A web application made with ReactJs

## :rocket: Getting Started

1. Download the repository

  - Using Git
```shell
  git clone https://github.com/jpalvesl/be-the-hero.git
```

2. Installation
```shell
  yarn
  npm install
```

3. Execution
```shell
  yarn start
  npm run start
```
