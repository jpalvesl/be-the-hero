## :clipboard: Description
An api made with Nodejs.

## :rocket: Getting Started

1. Download the repository

  - Using Git
```shell
  git clone https://github.com/jpalvesl/be-the-hero.git
```

2. Installation
```shell
  yarn
  npm install
```

3. Execution
```shell
  yarn dev
  npm run dev
```
